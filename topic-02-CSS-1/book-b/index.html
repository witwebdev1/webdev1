Practical Work

The purpose of this practical session is to get comfortable with using CSS style rules and linking them to HTML pages.

Practical One

  1. Open Practical One.
  2. Create a sub folder called css.
  3. Open your text editor and create a blank document. Save the file as cvstyle.css in the css sub folder.
  4. Add the following rules to the .css file (cvstyle.css in the css sub folder):
    1. body: background colour: hex FFFFE0; font colour: teal; font family: Book Antiqua or serif.
    2. h1 and h2: font colour: navy.
    3. hr: border width:0px, font colour: navy, background colour: navy, height:6px.
    4. a: font weight: bold.
    5. a:link: font colour: hex 0000CD.
    6. a:visited: font colour: black.
    7. a:hover: text decoration: none.
    8. a:active: font colour: aqua, background colour: navy.
  5. In the Practical One folder open each .htm file: (cv.htm; education.htm; and work.htm) and add a link tag to the section that links the .css file.

Practical Two

  1. Open Practical Two.
  2. In the Practical Two folder, create a sub folder called css.
  3. Open your text editor and create a blank document. Save the file as gardens.css in the css sub folder.
  4. In the Practical Two folder open index.htm and add a link tag in the section that links the .css file.
  5. Add the following rules to the .css file and in this example use ems for relative measurements (not percentages).
    1. body: background colour: hex EEEEEE; hex colour: 000080; font family Verdana or sans-serif, size: 15px;
    2. h1: background colour: hex BDB69C; colour: hex 565B60; increase the font size by 3 times the parent value; all uppercase.
    3. p: text alignment: justify and line height: 1 and half times the parent value.
    4. q: font family: Comic Sans or cursive; colour: DarkSlateBlue; increase the font size by 10% of the base not the parent.
    5. dt: font weight: bold and increase the font size by 10% of the parent.
    6. a: font weight: bold.
    7. a:link and a:visited: colour: navy and increase the font size by 10% of the parent.
    8. a:hover: text decoration: no underline; colour: olive.
    9. a:active: background colour: navy and colour: yellow.
