CSS Part One Exercises

Exercise One

  1. Open Exercise One folder.
  2. In the Exercise One folder, create a sub folder called css.
  3. Open your text editor and create a blank document. Save the file as kitchen.css in the css sub folder.
  4. In the Practical One folder open the index.htm file and add the following link tag to the <head> section:
    <link rel ="stylesheet" href="css/kitchen.css">
  5. Add the following CSS rules to the kitchen.css file in the css folder:
    1. body: font family: Verdana or sans-serif.
    2. h1: font family: Georgia or serif, font style: italic, font size: 3em, colour: lightseagreen.
    3. img: margin right: .5em.
    4. small: colour: slategray.

Exercise Two

  1. Open Exercise Two folder.
  2. In the Exercise Two folder, create a sub folder called css.
  3. Open your text editor and create a blank document. Save the file as exercise.css in the css sub folder.
  4. In the Exercise Two folder open index.htm and add a link tag in the <head> section that links the .css file.
  5. Add the following rules to the .css file:
    1. body: background colour: (E0FFFF); color: navy; font family: Garamond or serif.
    2. h3: background colour: blue; color: white; font size: 18px.
    3. p: font family: Verdana or Helvetica or sans-serif; font size: 14px; colour: #000080;
    4. Increase the size of the address element by 10%.
    5. Change the text inside the blockquote to uppercase.
